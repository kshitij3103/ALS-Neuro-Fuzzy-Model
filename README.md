# ALS Severity Prediction using Machine Learning

This project predicts ALS disease severity using clinical and respiratory indicators.

## Features Used
- ALSFRS functional scores
- Demographic features
- Forced Vital Capacity (FVC)

## Workflow
1. Data preprocessing
2. Exploratory Data Analysis
3. Feature engineering
4. Model training
5. Model evaluation

## Results

| Dataset | R² Score |
|-------|-------|
ALSFRS | 0.905 |
ALSFRS + Demographics | 0.906 |
ALSFRS + Demographics + FVC | 0.952 |

Raw datasets are not included due to GitHub file size limits.
